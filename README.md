# CleanSweep Website

Static GitHub Pages website for CleanSweep.

## Upload to GitHub

Upload everything in this folder to the `main` branch:

- `index.html`
- `privacy.html`
- `terms.html`
- `support.html`
- `contact.html`
- `delete-account.html`
- `README.md`
- complete `assets/` folder

The final structure must keep `assets/styles.css`, `assets/main.js`, logo and screenshots inside the `assets` folder.
