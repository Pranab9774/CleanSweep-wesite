# CleanSweep Website

Static production-ready website for CleanSweep.

## Pages
- `index.html` — landing page
- `privacy.html` — privacy policy
- `terms.html` — terms and conditions
- `support.html` — support page
- `contact.html` — contact page
- `delete-account.html` — data deletion request page

## Deploy on GitHub Pages
1. Create a GitHub repository, for example `cleansweepapp-site`.
2. Upload all files from this folder.
3. Go to Settings → Pages.
4. Select branch: `main`, folder: `/root`.
5. Save.

## Deploy on Cloudflare Pages
1. Create a new Cloudflare Pages project.
2. Connect the GitHub repository.
3. Framework preset: None.
4. Build command: leave empty.
5. Output directory: `/`.

## Before going live
Replace `support@cleansweepapp.com` with your working support email if different.
Review Privacy Policy and Terms with a qualified legal professional before publishing.
